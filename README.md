## üìå Projects Included

### 1. üè® Hotel Management System
**Description:** A console application for a hotel that manages room bookings. Users can view available rooms, book rooms by ID, calculate charges based on the length of stay, and complete payments using various methods.
* **Key Features:**
    * Room listing with IDs and prices.
    * Room availability and dynamic occupancy tracking.
    * Booking for multiple nights.
    * Automatic bill calculation and summary generation.
    * Multiple payment options: Cash, Card, UPI.

### 2. üçΩ Online Food Delivery System
**Description:** Simulates an online food ordering platform. Customers can select items by ID, specify quantities, and receive a detailed, itemized order summary with the total bill.
* **Key Features:**
    * Structured food menu with item IDs and prices.
    * Quantity-based ordering logic.
    * Handling of basic stock/availability.
    * Detailed order summary and accurate billing.

### 3. üöÜ Railway Reservation System
**Description:** A simple railway booking application that allows users to book train tickets. The system features train selection by ID and seat reservation logic based on real-time availability.
* **Key Features:**
    * Train list with capacity and current seat availability.
    * Seat reservation system logic.
    * Fare calculation per seat/ticket.
    * Comprehensive booking summary display.

### 4. üç¥ Restaurant Ordering System
**Description:** Focused on the in-house dining experience, this project allows customers or staff to place orders by selecting menu items using IDs. It efficiently calculates the total bill for table service.
* **Key Features:**
    * Clean menu display with unique item IDs.
    * Order placement with quantity tracking.
    * Total bill generation logic.
    * Clean and professional bill summary.

### 5. üè¶ ATM Management System
**Description:** A menu-driven simulation of an Automated Teller Machine (ATM). It features essential security through PIN authentication and supports common banking transactions.
* **Key Features:**
    * Secure PIN-based authentication.
    * Core banking operations: Balance check, Cash deposit, and Cash withdrawal.
    * User-friendly, loop-driven transaction menu.
